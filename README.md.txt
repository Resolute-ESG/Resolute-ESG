# ESG Risk Assessment Tool (Live Prototype)

This is a prototype ESG risk assessment app for suppliers. It allows users to:
- Input up to 10 suppliers and their annual spend
- Get real-time ESG indicators (simulated for now)
- Download a full RAG-rated ESG report

## How to Run

1. Clone this repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

3. Run the app:

   ```
   streamlit run app.py
   ```
